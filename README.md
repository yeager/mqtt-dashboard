# MQTT Dashboard

MQTT topic dashboard for monitoring IoT devices.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install mqtt-dashboard
```

### Fedora/RPM
```bash
sudo dnf install mqtt-dashboard
```
